## What I Learned This Month
- How an ECG signal travels from the heart to the display screen
- The function of each stage: electrodes → instrumentation amplifier → filters → isolation → ADC → microcontroller → display
- Why patient safety (isolation stage) is critical in medical device design
- Basic Arduino simulation: reading a sensor signal and triggering an LED

## Reflection
Understanding the ECG signal chain helps me see where failures happen in real hospitals—broken electrodes, faulty amplifiers, or power supply issues. This is directly relevant to Mali's medical equipment challenges.
